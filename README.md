To enable this plugin, you need to add these values to the nerdie config:

```
{
    "plugins": {
        "notify": {
                "announce": {
                        "reponame": ["#channel1","#channel2"]
                },
                "bitly_username": "<bitly username>",
                "bitly_apikey": "<bitly API key>"
        }
    }
}
```
